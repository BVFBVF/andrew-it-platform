# IT Start Platform — Задание

Стек: React, TypeScript, Next.js, Zustand, Tailwind CSS

Моковые данные придумай сам в src/lib/mock-data.ts

## Фича 1: Быстрый подбор под вакансию /match

- Вводишь требования: стек (теги), грейд,
  формат (удалёнка/офис), ставка
- Платформа показывает подходящих сотрудников
  отсортированных по readiness score
- На карточке каждого: совпадение по стеку в %,
  статус готовности, ставка
- Кнопка "Предложить клиенту" → копирует профиль
  сотрудника в буфер

## Фича 2: Воронка кандидатов /pipeline

- Список всех кандидатов с этапами:
  Новый → Задание выдано → Задание сдано →
  Готовится к собесу → Питчится клиентам →
  На собесе → Трудоустроен → Отвалился
- Карточка кандидата: имя, стек, грейд,
  текущий этап, дата последнего обновления
- Перемещение между этапами кнопками ← →
- Фильтр по стеку и грейду
- Счётчик кандидатов на каждом этапе

## Требования
- Компоненты декомпозированы
- Zustand для глобального стейта
- TypeScript, без any
- Адаптивная вёрстка
- Каждая фича — отдельная ветка и PR в main
